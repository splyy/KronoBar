01 — Vue principale

Ta journée,
en un coup d'œil

Le résumé de tes sessions s'affiche dès que tu ouvres KronoBar. Total d'heures, répartition par client, chaque entrée détaillée.

02 — Historique

Jour, semaine,
mois — ton choix

Installer sur macOS
1
Télécharger Télécharge le fichier .zip depuis le bouton ci-dessus.
2
Extraire Double-clique sur le .zip pour extraire KronoBar.app.
3
Déplacer dans Applications Glisse KronoBar.app dans ton dossier /Applications.
4
Débloquer Gatekeeper macOS va afficher un avertissement car l'app n'est pas signée avec un certificat Apple. Ouvre le Terminal et lance :
xattr -cr /Applications/KronoBar.app
5
Lancer Ouvre KronoBar — il apparaîtra dans ta barre de menus. C'est tout.
Installer sur Windows
1
Télécharger Télécharge le fichier Setup.exe depuis le bouton ci-dessus.
2
Lancer l'installateur Double-clique sur le .exe. Windows SmartScreen peut afficher un avertissement — clique sur "Informations complémentaires" puis "Exécuter quand même".
3
C'est prêt KronoBar s'installe et se lance automatiquement. Un raccourci est créé sur le Bureau et dans le Menu Démarrer. L'icône apparaît dans la zone de notification (system tray).
